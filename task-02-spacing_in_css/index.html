Introduction to the Web

The web is a vast system of interconnected documents and applications accessed through the internet. It allows users to browse websites,
interact with online services, and share information worldwide

Key Components of the Web

1- Web Browser

  • Google chrome
  • Mozilla Firefox
  • Microsoft Edge
  • Safari

2- Web Technologies

  1. HTML (Hyper Text Markup Language) - Structure of web page
  2. CSS (Cascading Style Sheets) - Styling and layout
  3. JavaScript - Interactivity and dynamic content

3- Types of websites

  1. Static Websites (Built with only HTML & CSS)
  2. Dynamic Websites (Use databases and server-side scripting)
  3. E-commerce Websites (Online shopping platforms)
  4. Blogs & Portfolios
